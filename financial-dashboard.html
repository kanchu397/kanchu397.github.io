An end-to-end BI solution for a family business, transforming scattered data into a strategic asset.

Live Dashboard

Project Deep Dive

The Challenge: The business's financial records were fragmented across 69 different Excel sheets. This made it impossible to get a quick, accurate overview of performance, identify top customers, or track cash flow effectively. All analysis was manual, time-consuming, and prone to errors.

My Solution: I tackled this by creating a full ETL (Extract, Transform, Load) pipeline. First, I wrote a Python script using the Pandas library to automatically extract the data from every sheet, clean it, and consolidate it into a single master CSV file. With this clean dataset, I then designed and built this interactive dashboard in Power BI.

Key Insights & Impact:

  • Identified Key Revenue Drivers: The dashboard immediately highlighted that a small number of "school" clients were responsible for a significant portion of revenue, suggesting a focus on this segment.
  • Improved Cash Flow Visibility: The trendline chart allowed the owner to visualize seasonal peaks and troughs in debit and credit, helping with better financial planning.
  • Enabled Data-Driven Decisions: By filtering by customer, the owner could instantly check account receivables, leading to faster payment follow-ups.
